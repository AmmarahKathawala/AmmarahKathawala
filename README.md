## MSc Dissertation (in progress)
**Machine Learning-Based Peak Quality Scoring in LC–MS Metabolomics Data**

Developing a data-driven framework to assess peak quality in LC–MS datasets, with the aim of improving the reliability and reproducibility of metabolomics workflows.

**Key components:**
- Quantitative peak feature extraction (intensity, area, width, signal-to-noise, peak symmetry)
- Machine learning for peak classification (Random Forest, SVM, Gradient Boosting)
- Benchmarking against traditional rule-based filtering
- Focus on robustness, generalisation, and scalability
